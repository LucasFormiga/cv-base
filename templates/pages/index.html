Sobre

Pyrocode é uma empresa de software focada em projetos web. Gostamos de trabalhar com ferramentas Python e JavaScript, mas temos uma mente aberta para experimentar e brincar com outras linguagens e ferramentas. Django e Vue são nossas especialidades e adoramos trabalhar com APIs.

Algumas tecnologias com as quais trabalhamos:

Django

um framework para aplicações web, de código aberto e gratuito, escrito em Python.!

Vue

framework JavaScript de código aberto, utilizando para construir interfaces e aplicações de página unica.

Python

é uma linguagem de programação de alto nível, orientada a objetos, funcional, de tipagem dinâmica e forte.

Node

é um ambiente de tempo de execução JavaScript de plataforma aberta, que executa o código JS fora de um navegador.
